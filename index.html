图片混淆/解混淆 - 反混淆分析版

🌐 API 调用（Vercel 云函数）

通过 Vercel 云函数处理图片，支持 API 调用。





原始算法核心逻辑（已还原）：

1. 把图片切成 N×N 个小方块（N = 等级数值，如4级 → 40×40 = 1600块）
2. 混淆（加密）步骤：
   a) 先把所有方块顺序完全颠倒（首尾互换）
   b) 用默认密钥 "hadsky.com" 做一次伪随机打乱
   c) 再用用户输入的密钥做一次伪随机打乱
3. 解混淆（解密）步骤（完全反向操作）：
   a) 先用用户密钥反向还原打乱
   b) 再用默认密钥反向还原打乱
   c) 最后把顺序完全颠倒回来
